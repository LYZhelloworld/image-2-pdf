# image-2-pdf
Compile your images into PDF files.

## Build
Output folder: `./bin`

### Linux
```
make

# To clean output folder
make clean
```
### Windows
```
build
```

## Test
### Linux
```
make test
```
### Windows
```
test
```

## Usage
```
img2pdf <directory that contains images> [<directory2 that contains images> ...]
```
